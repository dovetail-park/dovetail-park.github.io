## 2. 소스 코드
---
### 가. 텍스트 박스 ID
```vb
{
    Private Sub txtID_Enter()

       IF Me.txtID.Value = "아이디를 입력하세요."Then
          Me.txtID.Value = ""
          Me.txtID.ForeColor = RGB(51, 51, 51)
       End If
    End If
}
```
